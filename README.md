## рҹҺҜ The Problem

India loses **вӮ№120.3 crore annually** to "Digital Arrest" scams. Fraudsters impersonate CBI, ED, and RBI officers using:

- рҹҺҷпёҸ **AI voice cloning** вҖ” fake officer voices on WhatsApp calls
- рҹ“№ **Deepfake video** вҖ” fake police station backgrounds
- рҹ“„ **Forged documents** вҖ” fake FIRs, arrest warrants, court notices
- рҹ“һ **Spoofed caller IDs** вҖ” mobile numbers pretending to be government
- рҹ§  **Psychological pressure** вҖ” "digital arrest", "don't tell anyone"

**65% of victims are elderly. The Supreme Court of India has intervened.**

No existing tool detects all these attack vectors simultaneously. Until now.

## рҹӣ пёҸ Installation

### Prerequisites
- Python 3.10+
- Parrot OS / Ubuntu / Any Linux
- Webcam + Microphone
- AMD Ryzen AI NPU (optional, falls back to CPU)

### Setup
```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/sentinel-guard.git
cd sentinel-guard

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Install system tools
sudo apt install tesseract-ocr exiftool ffmpeg portaudio19-dev -y

# Launch SENTINEL-GUARD
python main.py
```
